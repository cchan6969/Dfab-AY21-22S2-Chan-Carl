Mewtwo


Pokemon chess set where Mewtwo chess piece was taken from here.
I have opted not to use the support provided by the file author but instead using the tree support generated by Cura as the tree support is easier to remove.
Mewtwo chess piece stl file
Mewtwo chess piece gcode

Inport of Mewtwo stl file into Cura

Settings used on Tevo Tarantula for Mewtwo Chess piece:

  • Layer Height: 0.1

  • Infill Density: 8%

  • Support Type: Tree

  • Build Plate Support: Brim

  • Print Time: Approximatly 3 hours
