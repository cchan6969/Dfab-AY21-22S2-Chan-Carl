3D printer

For school based printing, I have used the Ultimaker 2+ provided in school, and Tevo Tarantula printer at home.
The slicer program I have used is Ultimaker Cura 4.11.0.
With the Ultimaker 2+, I did a quick print of the aformentioned chess piece.
Settings used on Ultimaker 2+ for chess piece: 0.4 layer height, 8% infill density, tree support, brim build plate support. The print time was 10 mins.
Settings used on Tevo Tarantula for chess piece: 0.1 layer height, 8% infill density , tree support, brim build plate support. The print time was 45 mins
Pokemon chess set where Mewtwo chess piece was taken from here.
I have opted not to use the support provided by the file author but instead using the tree support generated by Cura as the tree support is easier to remove.
